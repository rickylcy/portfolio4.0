## Overview

A mobile ordering solution for restaurants that uses 2D barcodes at tables to start an order.

## Technologies

- React
- NextJS
- TailwindCSS
- Shadcn UI
- WebUSB Printing/ Windows driver printing
- NodeJS
- REST

## Tasks

- Build responsive UI with React + TailwindCSS + Shadcn UI
- Integrate backend REST APIs and design database structure
- Handle printings
- Prepare deployment pipeline
