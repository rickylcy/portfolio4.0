## 概覽

以 Next.js + Tailwind CSS 製作的電影瀏覽網站，提供熱門與高分分類，資料來自 TMDB。

## 技術

- Next.js
- Tailwind CSS

## 任務

- 使用 React + Tailwind 打造響應式 UI
- 串接 TMDB API
- 部署於 Vercel
