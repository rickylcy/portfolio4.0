可視化銷售數據的內部網站，提供響應式圖表與詳細報表。

## 技術

- React
- Material-UI
- JavaScript
- Chart.js
- HTML/CSS

## 任務

- 以 React + Material-UI 製作介面
- 串接 API 顯示即時數據
- 使用 Chart.js 建立互動圖表
- 部署應用
